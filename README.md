## Getting Started
Install the module with: `npm install onemap-crawl`

```javascript

onemapCrawl.awesome(); // "awesome"
```

```coffee-script
onemapCrawl = require 'onemap-crawl'
onemapCrawl.awesome() // "awesome"
```
